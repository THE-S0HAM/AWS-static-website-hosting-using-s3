Project Overview

This project demonstrates how to host a static website using Amazon S3. Amazon S3 is a powerful cloud object storage service by AWS that allows easy and scalable hosting of static web content. In this project, a simple multi-page HTML website is created and deployed on S3, showcasing personal info, documentation, and helpful resources. It guides you through creating a bucket, uploading files, setting permissions, and enabling static hosting â€” all without the need for a traditional web server. This is ideal for portfolios, documentation sites, and beginners learning cloud hosting.
